# URL Shortener 🔗

A modern and responsive URL Shortener web application built using **Flask**, **SQLite**, and **HTML/CSS**.

This project allows users to shorten long URLs, generate QR codes, track clicks, and instantly redirect to the original links.

---

## 🚀 Features

* Shorten long URLs instantly
* Unique short code generation
* QR code generation
* Redirect to original URL
* Click analytics support
* Copy-to-clipboard functionality
* Responsive modern UI
* SQLite database integration

---

## 🛠 Tech Stack

* Python
* Flask
* SQLite
* HTML/CSS
* QRCode Library

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/aishwaryaprakashg/url-shortener.git
```

### 2️⃣ Open Project Folder

```bash
cd url-shortener
```

### 3️⃣ Create Virtual Environment

```bash
python -m venv venv
```

### 4️⃣ Activate Virtual Environment

#### Windows

```bash
venv\Scripts\activate
```

### 5️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 6️⃣ Run Flask Server

```bash
python app.py
```
